Movies SE Prototype with Visualization

CPS 842 Final Project - By: Sakshi Padhiar & Majuth Kirakalprathapan

CTF-BM25-Normalized Search (Baseline Model)

The baseline model builds on BM25 by taking collection term frequency (CTF) into account. The rankings are also normalized. This search returns all relevant results sorted by highest ranking to lowest. It displays details such as Movie Title, Movie Overview, Genres and User Rating.


BM25 Search

The BM25 search returns all relevant results sorted by highest ranking to lowest. It displays details such as Movie Title, Movie Overview, Genres and User Rating.


Naive Bayes Search

The Naive Bayes search returns all relevant results sorted by highest ranking to lowest. It displays details such as Movie Title, Movie Overview, Genres and User Rating.


Compare All

This search displays visualizations for all 3 models (Baseline, BM25 and Naive Bayes) allowing them to be visually compared.
